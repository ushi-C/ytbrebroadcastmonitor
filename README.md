# YouTube转播监控器

🔗 **[点击这里直接使用](https://ushi-c.github.io/ytbrebroadcastmonitor/)**

在同一页面同时观看多路 YouTube 直播，体感人可以试听6个窗口已经上限了目前支持最多 6 个窗口。

---

## 快速开始

1. **打开页面** — 通过分享链接直接在浏览器打开，无需安装任何软件。
2. **添加播放窗口** — 点击右上角 **＋ 添加窗口** 按钮，页面会新增一个播放卡片。
3. **粘贴 YouTube 链接** — 在输入框中粘贴直播链接，支持以下格式：
   - `https://www.youtube.com/watch?v=xxxxx`
   - `https://youtu.be/xxxxx`
   - `https://www.youtube.com/live/xxxxx`
4. **点击播放** — 点击 **播放** 按钮，或直接按回车键，直播画面立即加载。

---

## 功能介绍

| 功能 | 说明 |
|------|------|
| 多窗口布局 | 支持单列、双列、三列切换，最多同时观看 6 路直播 |
| 拖拽换序 | 拖动窗口标题栏左侧的 ⠿ 图标，拖到另一窗口上松开即可交换位置 |
| 刷新播放 | 单个窗口点击 ↻ 刷新；顶部「刷新全部」按钮一键刷新所有窗口 |
| 关闭窗口 | 点击窗口右下角 ✕ 按钮关闭该播放卡片并释放资源 |

---

## 注意事项

- **仅支持 YouTube 链接**，其他平台链接无法播放。
- **直播内容需公开**，私密直播或地区限制内容无法加载。
- **窗口数上限为 6 个**，同时开启过多窗口会占用较多网络带宽，建议在网络条件良好时使用。
- **页面刷新后状态不保留**，所有窗口和链接需重新添加。
- **大量关闭窗口，调换窗口容易出bug**，针对该问题设计的界面内刷新，只是有所优化没有解决。
