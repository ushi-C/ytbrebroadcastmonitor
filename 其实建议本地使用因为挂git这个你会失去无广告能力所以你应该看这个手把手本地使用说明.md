#本地使用说明
---
先点那个code选择Download.zip。解压到新创建的文件夹例如你命名为ytbrbm的文件夹

## 第一步：确认电脑里有没有 Node.js

按键盘上的 **Win 键**（就是那个有 Windows 图标的键）+ **R 键**，同时按。

弹出一个小窗口，在里面输入：

```
cmd
```

按回车，会出现一个黑色窗口。

在黑色窗口里输入：

```
node -v
```

按回车。

- 如果出现类似 `v20.0.0` 这样的字 → **说明有，跳到第二步**
- 如果出现 `不是内部或外部命令` → **说明没有，需要先安装**

### 没有 Node.js 怎么安装？

1. 用浏览器打开 👉 https://nodejs.org
2. 点击页面上那个绿色大按钮（写着 LTS 的那个）
3. 下载完后双击安装包，一路点 **Next**，最后点 **Finish**
4. 安装完**重启电脑**
5. 重启后从第二步继续

---

## 第二步：把文件放到一个好找的地方

把 `index.html` 这个文件放到桌面或者某个文件夹里，**记住它在哪里**。

例如放在桌面上一个叫 `ytbrbm` 的文件夹里。

---

## 第三步：在文件夹里打开 PowerShell

在你放文件的**文件夹空白处**，按住 **Shift 键** 不放，同时**点鼠标右键**。

弹出的菜单里找 **"在此处打开 PowerShell 窗口"** 或 **"用 Terminal 打开"**，点它。

> 如果找不到这个选项，也可以这样做：
> 按 **Win 键 + X**，点 **PowerShell** 或 **终端**。
> 然后在黑色窗口里输入 `cd ` 再把文件夹拖进去，按回车。

---

## 第四步：启动本地服务器

在刚才打开的 PowerShell 窗口里，输入：

```
npx serve
```

按回车。

等一下，会出现一堆字，看到类似下面这样的内容就说明成功了：

```
Serving!
- Local:    http://localhost:3000
```

**这个窗口不要关！** 开着就行，最小化也行。

---

## 第五步：用浏览器打开

打开 Chrome 或 Edge 浏览器，在地址栏输入：

```
http://localhost:3000
```

按回车，就能看到播放器页面了。

---

## 第六步：开始看直播

1. 点右上角 **＋ 添加窗口**
2. 把 YouTube 直播链接粘贴到输入框里
3. 按回车或点 **播放**

最多可以同时开 6 个窗口。

---

## 每次用之前都要做什么？

每次想用都要重复 **第三步和第四步**，也就是：

1. 在文件夹里打开 PowerShell
2. 输入 `npx serve` 按回车
3. 浏览器打开 `http://localhost:3000`

---

## 常见问题

**Q：关掉 PowerShell 窗口后网页打不开了？**
重新打开 PowerShell，输入 `npx serve`，再刷新浏览器就行。

**Q：3000 端口被占用，提示换了端口怎么办？**
看 PowerShell 里显示的地址是什么，换成那个地址在浏览器里打开就行，比如 `http://localhost:3001`。

**Q：视频加载不出来？**
检查一下 YouTube 链接是否正确，以及该直播是否公开可见。

其他问题问AI或者作者，反正作者也是问AI。
